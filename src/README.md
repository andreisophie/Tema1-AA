# Tema 1 AA

by Andrei Maruntis

## Structura teste

Testele sunt grupate cate 5, cu numarul de noduri egal cu:

- 10
- 50
- 100
- 500
- 1000

- Teste 1 - 5:
  - orientate
  - rare (D = 0.2)
- Teste 6-10
  - orientate
  - densitate normala (D = 0.4)
- Teste 11-15
  - orientate
  - dense (D = 0.8)
- Teste 16 - 20:
  - neorientate
  - rare (D = 0.2)
- Teste 21-25
  - neorientate
  - densitate normala (D = 0.4)
- Teste 25-30
  - neorientate
  - dense (D = 0.8)
